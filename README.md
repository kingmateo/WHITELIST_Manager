## Installation Guide

### Step 1: Download and Extract
Download the WHITELIST release package from the Releases section. Extract all contents to a folder on your C: drive (or your Windows installation drive).

### Step 2: Disable Security Software (Recommended)
For optimal installation and first-time setup:
- Temporarily disable Windows Defender real-time protection
- Disable any third-party antivirus software
- Turn off Windows Firewall

### Step 3: Run as Administrator
Right-click on `WHITELIST_Manager.exe` and select **"Run as administrator"**. This is mandatory for the application to function correctly.

### Step 4: Activate Your License
- When prompted, enter the license key you received from Telegram
- Use Ctrl+V to paste the key if needed
- Click the "Activate" button

### Step 5: Configure V2Ray
Ensure the included V2Ray core (v2rayN-windows-64-v7.12.5) is in the same directory as the WHITELIST Manager. The application will automatically detect and use it.

### Step 6: Start Connection
- Click the "START SPOOFING" button
- Monitor the connection status in the real-time log panel
- The connection will remain active until you click "STOP"

## Troubleshooting Common Issues

**Issue: "Cannot fetch license" error** - This indicates a connection problem. Check your internet connection and ensure Windows Firewall is not blocking the application. Temporarily disabling the firewall usually resolves this issue.

**Issue: License expired message** - Your 7-day period has ended. Contact the Telegram channel to obtain a new license key.

**Issue: Connection unstable or not working** - Make sure you are running the application as Administrator. Also verify that the V2Ray core files are in the correct directory and that your antivirus software is not blocking the application.

**Issue: Application won't start** - Check that you have Python 3.14 installed. If not, download and install it from the official Python website. Also ensure you are running on Windows 10 or 11 64-bit.

**Issue: Log not saving or files not creating** - The application needs write permissions in its directory. Run as Administrator and ensure the folder is not read-only.
